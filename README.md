# Google Form Login

Ce projet est une page de login simple utilisant HTML, CSS, JavaScript et Bootstrap.

## Structure du projet
- `index.html` : Page principale avec le formulaire de login
- `style.css` : Styles personnalisés
- `script.js` : Logique JavaScript pour le formulaire

## Démarrage
Ouvrez `index.html` dans votre navigateur pour voir la page de login.

## Librairies utilisées
- [Bootstrap 5](https://getbootstrap.com/)

## Exemple de connexion
- Utilisateur : `admin`
- Mot de passe : `password`

Remplacez la logique de connexion par une vraie authentification pour un usage réel.
